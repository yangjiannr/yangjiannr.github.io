Posted on 2018-09-18 | In Distributed | 次阅读

编程环境搭建

MacOS

通过brew安装：brew install zmq

如果使用c++语言编程还需要获取到
zeroMQ的c++版头文件
git clone https://github.com/zeromq/cppzmq.git
将zmq.hpp拷贝到/usr/local/include下

Linux

如果使用c++语言编程还需要获取到
zeroMQ的c++版头文件
git clone https://github.com/zeromq/cppzmq.git
将zmq.hpp拷贝到/usr/local/include下
